
# Weather App - Chile

Aplicación web que muestra el clima de distintas ciudades de Chile.

## Características
- Listado de ciudades
- Vista de detalle
- Pronóstico semanal
- Diseño responsive

## Tecnologías
- HTML5
- CSS3
- Bootstrap
- JavaScript

## Cómo usar
Abrir el archivo index.html en el navegador.

## Autor
Proyecto desarrollado para módulo 2 Front-End.
